## About Me

我是 **Cocoon-Break**，最早公开的项目主要集中在前端、Vue 和 uni-app / 微信小程序方向：
[`neteaseMusic`](https://github.com/kuishou68/neteaseMusic) `147★`、
[`Vuejs`](https://github.com/kuishou68/Vuejs) `46★`、
[`hiolabs`](https://github.com/kuishou68/hiolabs) `17★`、
[`fruitsShop`](https://github.com/kuishou68/fruitsShop) `11★`。

最近这段时间，我把重心从前端单点开发扩展到 **AI 工具链、SaaS 产品和 AWS 上的基础设施落地**。对外可见的高 Star 参与项目里，最有代表性的是 [`iOfficeAI/AionUi`](https://github.com/iOfficeAI/AionUi) `18k★+`，我持续参与了 Codex / Gemini / Claude / 本地 agent 相关的模型接入、设置页、构建和稳定性修复。同时也在独立维护自己的 **Cocoon** 产品矩阵：`Home`、`API`、`SMS`、`VPN` 与社区化入口。

- Front-end / Vue / uni-app / mini-program background
- Contributor to `iOfficeAI/AionUi`
- Building Cocoon products and communities on AWS
- Writing on CSDN, Juejin, Zhihu, Bilibili, Blog
